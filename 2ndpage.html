control statements:

Control Statements are used to specify the flow of the program. C program executes sequentially from the first to last line. First, execute the first 
                statement, then second, third and so on. There are three types of control statements: 
                


                Decision Control
                Iterative
                Jump Statements
                Decision control: It checks whether the condition is satisfied or not. If the condition is satisfied then the statements are executed 
                and if the condition
                is false then it jumps to another condition. Decision control statements have four major types:
                1.If statement
                2.If else statement
                3.If else if statement
                4.Switch-case statement

               
                
               If Statement:It is a simple decision control statement
                Syntax:
                
                    If(Condition){
                         Block of code
                    }
                    Statement x;

                
                
                    If(printstatement){
                         Block of code
                    }
                    Statement x;
                
                


                If else Statement:The if-else statement in C is mostly used to perform operations based on some specific condition.
                Syntax:
                                If(condition){
                         Statement 1; 
                         }
                         else{
                         Statement 2;
                         }
                         Statement x;

                 
                
                Explanation: In an if-else statement, if the first condition is true, then statement 1 is executed and statement 2 is skipped. If the condition is false 
                then statement 2 is executed and statement 1 is skipped. And statement x is executed in both the conditions whether true or false.
                
                
                
                
                
                If else If statement: It is an extended version of the “if-else statement”. It checks additional conditions apart from the initial conditions
                Syntax:
                  if(condition 1)
                      {
                         Statement 1;
                  
                      }  else if(condition 2)
                      {
                         Statement 2;
                      }  else
                      {
                         Statement x;
                       } Statement y;
